# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report
it responsibly.

Do not publicly disclose vulnerabilities until they have been reviewed.

## How to Report

1.  Open a private security report if available
2.  Provide a clear description of the issue
3.  Include steps to reproduce the vulnerability
4.  Provide potential impact details

## Response

Project maintainers will review the report and determine appropriate
actions.

Security updates will be released when necessary.
