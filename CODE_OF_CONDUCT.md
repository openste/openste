# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and respectful environment for
everyone participating in the OpenSTE project.

## Expected Behavior

Participants are expected to:

-   Be respectful and considerate
-   Provide constructive feedback
-   Focus on improving the project
-   Respect differing opinions and experiences

## Unacceptable Behavior

The following behavior is not acceptable:

-   Harassment or discrimination
-   Personal attacks
-   Offensive language
-   Disruptive or hostile behavior

## Enforcement

Project maintainers are responsible for clarifying and enforcing these
standards.

They may remove or reject contributions that violate this Code of
Conduct.

## Reporting Issues

If you experience or witness unacceptable behavior, please report it by
opening an issue or contacting the maintainers.
