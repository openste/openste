# Examples

The **examples** directory contains sample documentation written using
OpenSTE guidelines.

These examples demonstrate how to apply writing rules and vocabulary in
real documentation.

------------------------------------------------------------------------

## Purpose

Examples help users understand how to:

-   simplify complex sentences
-   structure procedures
-   apply controlled vocabulary
-   write consistent technical instructions

------------------------------------------------------------------------

## Types of Examples

Examples may include:

-   installation instructions
-   user guide sections
-   maintenance procedures
-   troubleshooting steps
-   simplified documentation comparisons

------------------------------------------------------------------------

## Example

**Before**

    Ensure that the device is properly configured prior to initiating the operation.

**After**

    Configure the device before you start the operation.

------------------------------------------------------------------------

## Example Structure

Example files may include:

    installation_example.md
    maintenance_example.md
    troubleshooting_example.md

------------------------------------------------------------------------

## Contributing

You can contribute examples by:

-   rewriting complex documentation
-   creating sample procedures
-   demonstrating vocabulary usage
-   providing before/after comparisons

Examples should follow the guidelines defined in the `guidelines/`
directory.
