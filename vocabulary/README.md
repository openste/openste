# Vocabulary

The **vocabulary** directory contains the controlled vocabulary used by
OpenSTE.

A controlled vocabulary ensures that writers use consistent terminology
across all documentation. This improves clarity and prevents confusion
caused by multiple words describing the same concept.

------------------------------------------------------------------------

## Purpose

The vocabulary list helps:

-   standardize terminology
-   avoid ambiguous wording
-   simplify translation
-   maintain consistency across documentation

------------------------------------------------------------------------

## Vocabulary Rules

When writing documentation:

-   use approved words from the vocabulary list
-   avoid synonyms when a preferred term exists
-   keep terminology consistent across documents

Example:

    Approved: start
    Avoid: launch
    Avoid: initiate

------------------------------------------------------------------------

## Structure

Typical files in this directory may include:

    approved_terms.md
    restricted_terms.md
    terminology_rules.md

------------------------------------------------------------------------

## Example Entry

Example vocabulary entry:

    Term: start
    Part of speech: verb
    Meaning: begin operation of a system or device
    Example: Start the system.
