# Guideline — Limit “-ing” verb forms

## Guideline ID
1.0

## Guideline Title
Limit “-ing” verb forms to technical terms.

## Purpose
“-ing” forms can make sentences unclear or too long. Use simpler verb forms in instructions.

## Guideline
- **Do not** use “-ing” to describe actions in instructions.
- Use an “-ing” form **only** when it is part of a **technical term**, such as:
  - a **heading/title** (technical noun), or
  - a **technical noun phrase** (function modifier).

## Allowed uses (examples)
- **Headings/titles:** Cleaning, Testing, Packaging, Shipping, Troubleshooting
- **Technical terms:** air-conditioning system, grinding wheel, sanding machine, welding torch

## Preferred rewrite pattern
- **Avoid:** “When you are *doing* this procedure, obey all safety precautions.”
- **Prefer:** “When you do this procedure, obey all safety precautions.”
